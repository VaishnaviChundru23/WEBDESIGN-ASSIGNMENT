My Works

CLASSIFICATION USING MICRO-ARRAY DATA

Operate on various classification algorithms on microarray datasets where the main problem with this dataset is its dimensionality.Applied 3 optimization techniques (PCA, ICA and MI) on models, namely KNN, Random Forest Classifier and Decision Tree.

LEAKY INTEGRATE AND FIRE NEURON MODEL LATENCY ESTIMATION USING FNS

Using the neuron simulator FNS (Firenet NeuroScience), we estimate the latency of neuron model with different parameters. The artificial neural data using the parameters like number of neurons, rewiring probability and excitatory ratio is generated using the config wizard which provides an xml file. Feeding that xml file to the FNS simulator we get csv files which are used to visualize the raster plots in MATLAB.
